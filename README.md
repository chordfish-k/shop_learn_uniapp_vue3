# 小兔鲜儿商城小程序

跟着黑马做的仿电商程序，目前版本仍依赖黑马的后端数据

详情见[该项目的公共笔记](https://megasu.gitee.io/uni-app-shop-note/rabbit-shop/home.html)

## 项目信息

- 技术栈：vue3 + uni-app + TypeScript
- 开发环境：HBuilderX、vscode (主要插件: Volar, uni-helper, uni-create-view, uniapp小程序扩展)
